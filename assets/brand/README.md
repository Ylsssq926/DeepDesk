## 品牌设计：独角鲸小助手 (Chibi Narwhal Helper)

`logo.png` / `logo.svg` 采用了**独角鲸 (Narwhal)** 形象：
- **形象定义**：独角鲸是“海洋中的独角兽”（DeepSeek 的海洋/鲸鱼调性衍生），头顶的小角代表“智能天线、创作笔尖与深海探寻（Seek）”，非常契合第三方桌面增强外壳的辅助工具定位。
- **视觉风格**：萌系卡通、扁平化高亲和力（“讨喜”），与官方抽象、商务、高冷的侧面鲸鱼剪影形成鲜明区分，避免侵权和误导。
- **色彩规范**：采用柔和明亮的蓝青渐变底色，搭配干净的浅灰/白背景，在 16×16 等小尺寸下依旧清晰可读。

## 设计规范

参考 [`docs/UI-SKILL.md`](../../docs/UI-SKILL.md) 中的色彩与品牌规范：
- 主色：蓝青白渐变（hue 215 → 200）
- 不使用紫色（避开 AI 行业蓝紫审美撞车）
- 不模仿 DeepSeek 官方的抽象线条鲸鱼元素
- 保持简洁、高辨识度，确保缩放到 16×16 依旧清晰
