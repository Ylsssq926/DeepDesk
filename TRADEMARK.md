## 1. Why this document exists / 为什么有这份文档

The source code of DeepDesk is licensed under **AGPL-3.0**, which gives you
broad rights to use, study, modify, and redistribute the code. This document
describes the ONE thing AGPL-3.0 does **not** automatically give you: rights
to use the **DeepDesk name and logo**.

DeepDesk 的源代码采用 **AGPL-3.0** 协议授权，赋予你自由使用、学习、修改、分发
代码的权利。本文档说明 AGPL-3.0 **不**自动授予你的一项权利：**DeepDesk 名称
与 Logo 的使用权**。

## 2. What "DeepDesk Brand" means / 什么是 "DeepDesk 品牌"

The DeepDesk Brand includes, without limitation:

- The word "DeepDesk", "Deep Desk", or any variation that creates confusion
  with the DeepDesk project (e.g. "DeepDesk Pro", "DeepDeskAI", "DeepDesk
  Plus", "DeepDeskOfficial", "DeepDesk 官方版", etc.)
- The DeepDesk logo, icon, splash screen, and any visual mark, in any color
  or stylization, used to identify the DeepDesk project
- The "DeepDesk" wordmark in any font, language, or transliteration
- Any domain that materially uses "deepdesk" as the dominant element

These are collectively the **"DeepDesk Brand"** and are NOT covered by
AGPL-3.0. They are owned and reserved by the DeepDesk project maintainers.

DeepDesk 品牌包括但不限于：
- 文字「DeepDesk」、「Deep Desk」或任何使读者与 DeepDesk 项目产生混淆的变体
  （如 "DeepDesk Pro"、"DeepDeskAI"、"DeepDesk Plus"、"DeepDeskOfficial"、
  「DeepDesk 官方版」等）
- DeepDesk 的 Logo、图标、启动屏及任何视觉标识
- 任何字体、语言或音译形式的"DeepDesk"文字标志
- 任何以"deepdesk"为主导元素的域名

The following uses are **automatically permitted** without prior authorization,
provided they are **truthful, non-misleading, and non-commercial-promotional**:

以下用途**自动授权**，无需事先许可（前提：真实、不误导、非商业推广）：

- ✅ Mentioning DeepDesk in articles, blog posts, tutorials, videos,
  conference talks for **educational, comparative, or review** purposes.
  在文章、博客、教程、视频、演讲中**因介绍、对比、评测**目的提及 DeepDesk。

- ✅ Using the name "DeepDesk" to truthfully describe a fork's relationship
  to the original project, e.g. "Forked from DeepDesk" or "Based on
  DeepDesk", **provided your fork itself is renamed**.
  使用「DeepDesk」一词如实描述 fork 与原项目的关系（如 "Forked from
  DeepDesk"），**前提是 fork 本身已改名**。

- ✅ Linking to the official repository at
  https://github.com/Ylsssq926/DeepDesk.

- ✅ Using the DeepDesk logo verbatim and unaltered in articles or videos
  about DeepDesk, with attribution.
  在介绍 DeepDesk 的文章/视频中**原样、未修改**使用其 Logo，并附署名。

- ✅ Internal use within an organization without redistribution to third
  parties (e.g. one company internal experiment).
  组织内部使用，不向第三方分发。

## 5. Mandatory Renaming for Forks / Fork 必须改名

If you redistribute a modified version of the DeepDesk source code under
AGPL-3.0, you **must rename your derivative work** so it is not confused
with DeepDesk. The new name should:

如果你以 AGPL-3.0 协议分发 DeepDesk 源代码的修改版，**必须为衍生作品改名**，
避免与 DeepDesk 混淆。新名应：

1. Not contain "DeepDesk" as the primary identifier (a small "based on
   DeepDesk" attribution is fine, but the product name itself must differ).
   不以「DeepDesk」为主标识（在 README 中标注「基于 DeepDesk」可以，但产品
   名本身必须不同）。
2. Not use the DeepDesk logo or any visually similar mark.
   不使用 DeepDesk Logo 或任何视觉相似标识。
3. Not impersonate "official", "verified", or "endorsed" status.
   不冒充「官方」「认证」「授权」身份。

Examples of acceptable forks (hypothetical):
可接受的 fork 命名示例（虚构）：
- ✅ "MeowChat — based on DeepDesk"
- ✅ "FoxAI Desktop (an unofficial fork of DeepDesk)"
- ✅ "深思助手 — 基于 DeepDesk 的社区分支"

Examples of NOT acceptable:
**不**可接受的命名示例：
- ❌ "DeepDesk Pro" / "DeepDesk Plus" / "DeepDesk Premium"
- ❌ "DeepDesk 官方版" / "DeepDesk Official"
- ❌ "DeepDesk Mini" / "DeepDesk Lite"
- ❌ Any logo that is visually similar to the DeepDesk logo

This is consistent with the practice of Mozilla / Firefox (renamed forks:
IceCat, IceWeasel) and Chromium / Brave.
此规则与 Mozilla / Firefox（fork 改名为 IceCat、IceWeasel）以及 Chromium /
Brave 的常见做法一致。
