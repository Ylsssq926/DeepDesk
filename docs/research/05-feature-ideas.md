# DeepDesk 特色功能候选库（头脑风暴）

> 时间：2026-03
> 范围：在 PRD 已确定的 MVP 11 项 + V0.2 11 项 + V1 3 项 + V2 7 项之外，穷举可加入 V0.2 / V1 / V2 / Backlog 的特色功能。
> 原则：「为 DeepSeek 量身定制 + 查漏补缺 + 优雅退场」——补缺，不重复，可一键关。
> 字段约定：
> - **层**：L1 系统集成 / L2.A 长线补丁 / L2.B 短窗口补丁 / L3 精修
> - **DOM 依赖**：是 = 对官方 DOM 选择器/类名有强依赖，易随改版失效；否 = 仅依赖 SSE 拦截 / 本地数据 / OS 能力
> - **合规风险**：无/低/中/高 + 简短理由
> - **价值评分**：⭐ 1-5 颗（用户呼声 + 差异化强度 + 黏性增加）

---

## 一、会话 / 对话维度（"二脑"工具）

### C-01 对话文件夹 + 嵌套分组 + 颜色编码
- **描述**：在 F-08 本地标签的基础上叠加多级文件夹、彩色标签、自定义图标，侧栏可折叠展开。
- **层**：L2.A
- **版本**：V1（在 V1-02 知识库内深化）
- **价值**：⭐⭐⭐⭐
- **难度**：中
- **DOM 依赖**：否（独立侧栏，不动官方左栏）
- **合规风险**：无（纯本地元数据）

### C-02 消息片段卡（Snippet Card）
- **描述**：收藏单条消息（不只是整个对话），独立成卡，支持标签、笔记、跨对话引用。
- **层**：L2.A
- **版本**：V1
- **价值**：⭐⭐⭐⭐⭐
- **难度**：中
- **DOM 依赖**：否（拦截 SSE 已有完整消息 ID）
- **合规风险**：无

### C-03 跨对话 @ 引用
- **描述**：在新对话输入 `@` 触发器，可检索并插入历史对话中某条消息作为上下文（明确标注来源）。
- **层**：L2.A
- **版本**：V1
- **价值**：⭐⭐⭐⭐⭐
- **难度**：中
- **DOM 依赖**：低（只在输入框层做 autocomplete，不动消息列表）
- **合规风险**：无（用户自己输入，等价于复制粘贴）

### C-04 对话快照 / 时光机
- **描述**：用户可在对话任意节点"打点"，随后可一键时光回溯到该节点重新分支。
- **层**：L2.A
- **版本**：V1
- **价值**：⭐⭐⭐
- **难度**：中
- **DOM 依赖**：否（基于本地存档的消息序列）
- **合规风险**：无

### C-05 对话差分（Diff View）
- **描述**：选中两个对话/两段回答 → 并排 diff 视图（用 Monaco DiffEditor），便于对比模型不同版本/不同提示的输出。
- **层**：L2.A
- **版本**：V1
- **价值**：⭐⭐⭐
- **难度**：低
- **DOM 依赖**：否
- **合规风险**：无

### C-06 自动对话标题（AI Auto-Title）
- **描述**：新对话产生第一条回答后，本地用一次轻量请求让 DeepSeek 生成 8-15 字标题，自动覆盖默认标题（用户可关）。
- **层**：L2.B
- **版本**：V0.2
- **价值**：⭐⭐⭐⭐
- **难度**：低
- **DOM 依赖**：中（需要把生成结果写入侧栏标题——可走"重命名 API"或仅在本地标签上覆盖）
- **合规风险**：低（额外占用一次配额，需提供开关）

### C-07 对话评分 + 心情标记
- **描述**：每个对话/回答可标 👍/👎/⭐ + 自定义 emoji，作为后续筛选/复盘维度。
- **层**：L2.A
- **版本**：V1
- **价值**：⭐⭐⭐
- **难度**：低
- **DOM 依赖**：否
- **合规风险**：无

### C-08 对话克隆
- **描述**：右键任意对话 → "克隆为新对话"，本地把消息序列重放为新对话的起点（首条消息为合并版上下文）。
- **层**：L2.A
- **版本**：V1
- **价值**：⭐⭐⭐
- **难度**：中
- **DOM 依赖**：低（仅写入输入框）
- **合规风险**：无

### C-09 多回答批量再生
- **描述**：在历史回答上多选 → "全部重新生成"，DeepDesk 串行触发官方"重新生成"按钮。
- **层**：L2.B
- **版本**：V1
- **价值**：⭐⭐
- **难度**：中
- **DOM 依赖**：是（依赖官方 regenerate 按钮）
- **合规风险**：低（产生额外配额，需明示）

### C-10 对话健康报告
- **描述**：每周/月自动生成一份"我的 DeepSeek 使用报告"（话题分布、时间段、最长 Thinking、被收藏次数 Top 5），仅本地。
- **层**:** L2.A
- **版本**：V1
- **价值**：⭐⭐⭐
- **难度**：低
- **DOM 依赖**：否
- **合规风险**：无
